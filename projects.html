Real Time Bus Tracker

With this Bus Tracker you can find all buses of one specific route. The tracker refreshes every 10 seconds so you have always the newest locations. This project receives the data from the MBTA with "fetch" and creates with that data the marker for each bus.

  • Choose between map styles
  • Choose between different routes
  • Actual data from the MBTA API
